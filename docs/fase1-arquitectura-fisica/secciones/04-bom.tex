\section{Lista de materiales (BOM) --- Fase 1}
\label{sec:bom}

Esta BOM incluye exclusivamente los componentes físicos del rack, los
contenedores y el subsistema de \textbf{esterilización e inoculación}
(\S\ref{sec:preparacion}). El BOM electrónico (sensores, ESP32, cámaras,
humidificadores, broker) se consolida en la \textbf{Fase~8} tras cerrar
las decisiones de las Fases~2 y~3.

\textbf{Política de precios:} ningún precio se inventa. Cada renglón
sin cotización aparece como \cotizar{...} y se replica en
\texttt{cotizaciones\_pendientes.md} con búsqueda sugerida por
proveedor mexicano (MercadoLibre MX, Home Depot, Steren, AliExpress,
Tornimex).

% ---------------------------------------------------------------------
\subsection{Estructura del rack}
\label{sec:bom:rack}

\begin{itemize}[leftmargin=*,itemsep=0pt]
  \item Rack metálico \paramNiveles{} niveles, $\sim$\paramHmax$\times$\paramWmax$\times$\paramDmax\,cm,
        carga $\geq$\paramCargaUtilNivel\,kg/nivel.
        \cotizar{anaquel metálico 5 niveles 180×90×45 cm Home Depot MX}
  \item Niveladores roscados M10 con goma (×4).
        \cotizar{niveladores patas M10 con goma Steren}
  \item Anclaje antivuelco a muro (kit con taquetes y solera).
        \cotizar{kit antivuelco anaquel Home Depot}
  \item Tapete antiderrame plástico para nivel inferior.
        \cotizar{tapete antiderrame plástico 90×45 cm MercadoLibre MX}
\end{itemize}

\subsection{Diferido a Fase 8 (electrónica)}
\label{sec:bom:diferido}

No cotizar hasta cerrar Fases 2 y 3:

\begin{itemize}[leftmargin=*,itemsep=0pt]
  \item Sensores SHT35, MH-Z19B, BH1750, DS18B20.
  \item Multiplexor I$^2$C TCA9548A (depende de topología \S2.2).
  \item ESP32-S3 (con cámara) o ESP32 + ESP32-CAM separados.
  \item Humidificadores ultrasónicos por contenedor.
  \item Ventiladores 5\,V/12\,V para FAE.
  \item LEDs y drivers para fotoperiodo.
  \item Raspberry Pi 4/5 o mini-PC dedicado al broker MQTT + TSDB.
\end{itemize}
